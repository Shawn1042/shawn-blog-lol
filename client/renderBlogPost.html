1. Lazy Loading Images

By implementing lazy loading, images are only loaded when they enter the viewport. This reduces the initial load time significantly.

2. Deferring Rendering of DOM Nodes

To avoid rendering all elements at once, we use the Intersection Observer API to render elements as they come into view. This optimizes the rendering process and improves performance.

const observer = new IntersectionObserver((entries) => {
entries.forEach(entry => {
if (entry.isIntersecting) {
const card = entry.target;
card.style.height = ''; // Remove placeholder height
observer.unobserve(card);
}
});
}, { rootMargin: '100px' });

3. Minifying CSS and JavaScript

Minifying CSS and JavaScript files reduces their size, leading to faster load times. This involves removing whitespace, comments, and unnecessary characters.


4. Using Efficient CSS Selectors

Efficient CSS selectors ensure that styles are applied quickly and do not cause delays in rendering.


5. Combining CSS and JavaScript Files

Combining multiple CSS and JavaScript files into single files reduces the number of HTTP requests, enhancing load times.
